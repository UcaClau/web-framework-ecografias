Deep Brain

Sobre el proyecto

Deep Brain es un framework basado en Deep Learning para el análisis automatizado de ecografías cerebrales neonatales. La presente página web pretende presentar y habilitar el uso de este software, capaz de realizar diferentes acciones sobre ecografías cerebrales. Este proyecto forma parte de un trabajo de fin grado (TFG) y tiene como objetivo principal facilitar el análisis de este tipo de imágenes médicas mediante técnicas de Deep Learning. Además, se busca proporcionar una herramienta útil para profesionales de la salud, ya sea como segunda opinión o para que se utilice como apoyo en la formación de nuevos profesionales.

Las funcionalidades principales del software son:

Clasificar

Segmentar

Generar datos
