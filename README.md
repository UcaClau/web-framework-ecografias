# Web del proyecto: Deep Brain
Esta web forma parte del Trabajo Fin de Grado presentado por Claudia García-Matarredona Urbano.

En ella se presenta una herramienta capaz de clasificar y segmentar ecografías, o ultrasonidos, tridimensionales de cerebros neonatales (recién nacidos), con el objetivo de asistir en el análisis clínico y la formación de nuevos profesionales de la salud.

Asimismo, se dispone de un manual de usuario que recoge tanto el funcionamiento, como el desarrollo y desempeño de las redes implementadas.

Cabe destacar, que este proyecto sigue en desarrollo, y que cualquier aportación es bienvenida.
